# VibeCoding - Start Page

이 프로젝트는 브라우저의 시작 페이지 또는 대시보드 화면으로 활용할 수 있는 웹 페이지입니다. 

## 📝 주요 기능
- **실시간 시계 (Clock)**: 현재 날짜와 시간을 실시간으로 보여주는 대형 시계 위젯을 제공합니다.
- **날씨 정보 (Weather)**: 사용자가 지정한 지역 또는 현재 위치의 날씨와 기온 정보를 보여주는 섹션이 포함되어 있습니다.
- **뉴스/정보 (News)**: 하단 그리드에 최신 뉴스나 업데이트된 정보를 표시할 수 있습니다.
- **반응형 디자인**: 화면 크기(PC, 태블릿, 모바일)에 맞춰 레이아웃이 자연스럽게 변경됩니다. (CSS Grid / Flexbox 적용)

## 🛠 사용 기술
- **HTML5**: 시맨틱 태그를 활용한 구조 설계
- **CSS3**: Linear Gradient, Box Shadow 등을 활용한 모던 UI 및 반응형 웹 디자인
- **Vanilla JavaScript**: 실시간 시간 업데이트 및 동적 데이터 렌더링

## 🚀 시작하기
별도의 서버 구축 없이, 브라우저에서 바로 `index.html` 파일을 열어서 확인할 수 있습니다.

1. 이 저장소를 클론하거나 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 크롬, 엣지 등의 웹 브라우저로 엽니다.
3. 원하는 위젯 데이터(날씨 API, 뉴스 API 등)를 연동하려면 자바스크립트 코드를 수정하여 사용할 수 있습니다.
